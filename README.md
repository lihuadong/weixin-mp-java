# 微信公众平台  weixinmp4j 
微信公众平台API封装 Java 版本

weixinmp4j，项目原名weixin4j

此项目为是更好的使用微信公众平台的功能而进行的JAVA版本的代码封装
在调用对应的方法之前需要传对应项目的Accesstoken 
建议在使用之前Accesstoken先进行缓存，存入DB或文件均可
weixin.base包中AccesstokenManager也可以供使用

一、项目说明
1、jdom 对xml数据进行处理
2、json 对json数据进行处理
3、log4j对日志数据进行处理
4、网络请求包
